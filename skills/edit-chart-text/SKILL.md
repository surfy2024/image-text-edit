# Edit Chart Text

Replace only the requested chart text and preserve all unrelated image content.

## Workflow

1. Identify the exact source text, replacement text, and input image.
2. Inspect the target region and typography before editing.
3. Apply the smallest possible localized edit.
4. Save the result separately from the original image.
5. Verify the replacement and compare unaffected regions with the source.
